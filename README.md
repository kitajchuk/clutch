### Getting started

Install the clutch command line interface (CLI):

```shell
npm i clutch-cli -g
```

Initialize your new clutch project:

```shell
# In cwd
clutch init

# Create new dir
clutch init my-project
```

Install the NPM dependencies and boostrap the scaffold:

```shell
npm run bootstrap
```
